Usamos a tag code para facilitar a leitura de códigos, pois ele deixa a fonte monoespaçada, veja um exemplo

O código document.getElementsbyId( 'teste) é escrito em JavaScript

Fonte monoespaçada significa que ela reserva o mesmo espaço para todos os caracteres, ou seja, uma letra l ocupa o mesmo espaço de uma letra E.

O problema é quando queremos mostrar um código com várias linhas. Apenas a tag code não resolve este problema, veja:

CITAÇÕES SIMPLES

Citação simples é aquele sem quebra de linha

Para inserir citações recomenda-se usar a tag q . Significa quotation e é uma tag semântica, por isso carrega mais significado do que simplesmente colocar as aspas. Exemplo:

O computador é um burro muito rápido Esta citação foi inserida usando a tag q

CITAÇÃO COM BLOCKQUOTE

Citaçao com blockquote é uma citação com quebra de linha, espaçamento e na qual é possível inserir a origem da citação

Exemplo

Segundo carol S. Dewck:

Acreditar que suas qualidades são imutáveis - o mindset fixo - cria a necessidade constante de provar a si mesmo seu valor. Se você [pssui apenas uma quantidade limitada de inteligência, determinada personalidade e certo caráter moral, nesse caso terá de provar a si mesmo que essas doses são saudáveis. Não lhe agradaria parecer ou sentir-se deficiente quanto a essas características fundamentais.

O parâmetro cite não aparece na página, ele é apenas um referência que indica para mecanismos de busca de onde a informação foi tirada
