Teste com a tag picture

A tag picture vai possuir dentro dela uma tag img, mas ela vai permitir diversos "sources" diferentes e isso vai permitir que o site carregue uma imagem de tamanho mais adequado a tela do usuário

Veja abaixo o exemplo

Primeiro, por padrão, o site carregará a imagem grande - foto-g.png

Porém, caso a tela possua largura até 1050px ele carregará a imagem Média

E se o tamanho de largura for menor do que 750px o site mostrará a imagem Pequena
