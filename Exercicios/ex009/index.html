Listas Ordenadas

Lista ordenada é um tipo de lista em que importa a sequência da lista ou uma lista simples que você queirar numerar sequencialmente, veja o exemplo:

  1. Acordar
  2. Tomar Café
  3. Escovar os dentes

É possível usar o parâmetro type para alterar o tipo de "numeração", ordenação. Veja o exemplo com o parâmetro numerado padrão:

  1. Primeiro item
  2. Segundo item
  3. Terceito item

LISTAS NÃO ORDENADAS

Listas não ordenadas são listas que não precisam ser numeradas, ou listas em que a sequencia não importa, como uma lista de compras por exemplo

  • Pão
  • Cerveja
  • Refrigerante

É possível mudar o parâmetro type para alterar o marcador da lista. Se não colocar nada, por padrã fica o círuclo fechado

São trës parâmetros de marcadores possíveis: Circle, Disc, Square , veja cada um deles:

Type="circle"

LISTA DE DEFINAÇÕES - tag "dl"

A lista de definições funciona como um dicionário, você adiciona um TERMO e depois adiciona uma descrição. O termo é o "DT" e a descrição é "DD". Assim:

HTML
Linguagem de marcação para criação de sites

CSS
Linguagem de marcação para criação do design de um site

JavaScript
Lingagem de programação para a criação de interatividade de um site
